How to compile:
javac *.java

How to run:
java HISS

How to use the program:
For player1, use keys W, A, S, and D to move the snake up, left, down, and right respectively. For player2, use keys I, J, K, and L to move the snake up, left, down, and right respectively. We have yet to implement the snakes' interaction with each other, with the items in the arena and with the walls.
