How to compile:
javac *.java

How to run:
java HISS

How to use the program:
For player1, use keys W, A, S, and D to move the snake up, left, down, and right respectively.
For player2, use keys I, J, K, and L to move the snake up, left, down, and right respectively.
Press P to pause the game
Mouse input used to click buttons to navigate help and menu screens
